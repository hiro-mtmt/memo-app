ラクメモ

Markdownが使える、シンプルで軽量なデスクトップメモアプリ。

macOS (Apple Silicon) macOS (Intel) Windows

Features

M↓

Markdown対応

GFM対応のMarkdownエディタ。リアルタイムプレビューで快適に書ける。

📁

フォルダ管理

メモをフォルダで整理。ドラッグ&ドロップで簡単に移動できる。

💾

自動保存

編集内容は自動で保存。データはすべてローカルに安全に保管。

🔍

ピン留め

よく使うメモをピン留めして、リストの上部に固定できる。

📝

.md / .txt 対応

MarkdownとテキストファイルどちらもOK。用途に合わせて選べる。

⚡

軽量&高速

Tauri + Rustで構築。メモリ使用量が少なく、起動が速い。

インストール方法

⚠ 注意: このアプリはコード署名されていません。初回起動時に警告が表示されますが、以下の手順で開けます。

macOS

  1. .dmg ファイルをダウンロード
  2. 開いてアプリを Applications フォルダへドラッグ
  3. 初回は右クリック → 「開く」を選択
  4. 確認ダイアログで「開く」をクリック

Windows

  1. .msi または .exe をダウンロード
  2. 実行し、SmartScreen の警告が出たら「詳細情報」→「実行」
